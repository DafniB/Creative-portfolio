# Creative Developer Portfolio

A modern **developer portfolio website** built using **HTML, CSS, and JavaScript**.

This portfolio showcases projects, skills, experience, and contact information with a clean modern UI.

---

## 🚀 Features

* Modern responsive design
* Smooth scrolling navigation
* Skills showcase section
* Project gallery
* Experience timeline
* Contact form UI
* Mobile friendly layout
* Clean UI/UX

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* Google Fonts
* Unsplash Images

---

## 📂 Project Structure

```
creative-portfolio
│
├── index.html
├── README.md
└── assets (optional for images)
```

---

## 🌐 Preview

A modern portfolio layout with sections including:

* Hero introduction
* About section
* Skills
* Projects
* Experience timeline
* Contact form

---

## ▶️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/yourusername/creative-portfolio.git
```

2. Open the project folder

3. Run the website by opening:

```
index.html
```

in your browser.
